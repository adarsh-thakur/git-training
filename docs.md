### Create A branch
- `git branch <branch-name>`
### Checkout a branch
- `git checkout <branch-name>`

OR
### Short Hand
`git branch checkout <branch-name>`


### stage the changes to be committed
`git add <file-name> or <.>

### commit your changes
`git commit -m "Commit Message"


## push your changes

git push -u origin <branch-name>
